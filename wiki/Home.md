# Agentic AI Demo - Wiki

Release 0.5 focuses on simpler, story-driven test prompts that can be run one by one.

## Pages

| Page | Purpose |
|---|---|
| 01 - Setup: Claude Code | Install, drag-and-drop, verification in the Agents center |
| 02 - Setup: GitHub Copilot | Install, drag-and-drop, verification in the Agent picker |
| 03 - Test Prompts L1 | Simple single-agent prompts plus expected outcomes |
| 04 - Test Prompts L2 | Multi-agent handoff prompts plus expected outcomes |
| 05 - Test Prompts L3 | Full-ecosystem tests exercising all agents and skills |
| 06 - Adding Agents & Skills | Templates and conventions for growing the roster |

## Release notes

- Prompts are smaller and easier to follow in a live demo.
- Every prompt is framed as a short story about an engineer with a task.
- The L1 and L2 paths now build up from small infra to full handoffs more gradually.

## Suggested flow

1. Run the setup page for the platform you are using.
2. Start with L1 and work through the prompts in order.
3. Move to L2 only after the single-agent tests are passing.
4. Use L3 when you want the full orchestrator and review gates.
