# Release 0.5

Release 0.5 simplifies the prompt set so it is easier to walk through in a live demo.

## What changed

- L1.1 and L1.2 are now smaller hub-and-spoke style landing-zone stories with a VM, an egress gateway, and a few network security controls.
- The rest of the prompts are written as short stories about an engineer and a task, so each test is easier to explain and run one by one.
- The L2 and L3 flows still exercise the same agent boundaries, but with less unnecessary complexity in the starting scenarios.

## Release note for demo runners

Start with L1, move to L2 after the single-agent prompts are green, and use L3 for the full ecosystem once the team is comfortable with the flow.
